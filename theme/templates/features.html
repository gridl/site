Features

Websauna provides secure, high level building blocks, without taking the driving seat from a developer. By having the basic needs and infrastructure of web applications catered you can focus on the business logic of your web application.

Admin

Admin, administration interface, provides drop in back office for your operations. You can easily browse, add and edit data without need to write explicit form code. SQLAlchemy data models are automatically turned to an admin site.

Documentation

Detailed tutorials and deployment instructions help developers to get their first application out fast. Comprehensive API documentation, template and configuration reference makes sure no corner goes undocumented.

Branding

Websauna comes with basic branding elements ready to go. Default templates use Bootstrap CSS framework that allows to choose from beautiful theming options even if the team lacks design skill. Email subsystem comes with well tested HTML templates to ensure brand integrity in messaging.

Jupyter Notebook

Jupyter Notebook, an award winning data analysis and science tool, is directly integrated to Websauna. Analyzing website data and building interactive visualizations is within a reach of one click. Furthermore web based Python shell provides alternative for a terminal for interactive Python development.

Optional typing

APIs use Python 3.5 type hinting systems. This allows IDEs and code editors auto complete and lint your Python code while you type. Statistical analysis tools can be run to check for errors.
